# InfiniteRunner

## 특징
 - 무한히 생성되는 맵을 죽지 않고 달려보자.
 - 획득한 코인을 소모하여 대시를 사용할 수 있다.
 - 까만 구체를 먹으면 일정시간 무적이다!
 
## 이동 방법
- w : 앞으로 가기 토글
- a,d : 좌우로 이동
- space bar : 점프
- f : dash

## 기능
대부분 블루프린트로 구현하였습니다.
